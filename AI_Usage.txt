I wanted to test Codex's abilities after what was taught in class. To do that, I added on to the AGENTS.md that we made in class to give it example code from my weather app project and code we had written in class. I added that example code to AGENTS.md by asking ChatGPT what the best way to add example code to my AGENTS.md would be and following its guidelines. I then gave ChatGPT a detailed prompt asking it to make a prompt for Codex with the context of what this project needed by reading though the project description and putting features into the prompt in my own words. I had it give me the prompts it generated in small chunks so each prompt given to codex would add only a few features at a time. After each time Codex wrote code, I would test the new features and all of the old ones to ensure they worked before moving to the next prompt. I wanted to do this until Codex started to fall apart and create a lot of bugs. However, Codex surprised me with how good the code it was writing was. Not only was it mostly bug free, but it was following my example code extremely well which kept the code very understandable because it was writing code very similar to how I do. I only had to fix a few bugs on my own, make the frontend look a little prettier, and add the icons. Almost everything else was agentically coded and then tested by me.
